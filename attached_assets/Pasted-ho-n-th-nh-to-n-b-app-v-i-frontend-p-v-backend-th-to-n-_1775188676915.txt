hoàn thành toàn bộ app, với frontend đẹp, và backend , thì toàn bộ flow như sau
project của t như này
- một cái lộ trình học, chia ra 2 mode, cùng dashboard và progress weekly update
mode exam thì chỉ cần biết đề thi sẽ ra topic gì, sau đó cho mini test, nếu nó yếu phần đó thì học những phần đó
còn nếu general study thì tạo lộ trình học dựa trên mục tiêu
input là : username, password (cái này phần signup/login), subject(1 trong 4 môn toán, lý, hoá, sinh), lớp (10 -> 12), MODE (general, hoặc exam)
đó là dữ liệu cứng user nhập ở input khi mới vào giao diện web (mỗi câu hỏi là 1 trang khác nhau nha, sẽ tự động chuyển trang
database câu hỏi, tạo thành mini test và sau khi làm xong, mình sẽ có data vè những topic mà user còn yéu để học
có thể dùng, tích hợp LLm vào để break down topic thành các subtopic đồng thời đưa ra lời khuyên chi tiết hơn và cá nhân ho
# 1. DATABASE SCHEMA (CHẠY HỆ THỐNG)
## (A) User
```sql
users(
user_id PK,
username,
password_hash,
grade
)
```
---
## (B) Learning Config
```sql
learning_configs(
config_id PK,
user_id FK,
subject,
mode, -- exam / general
target_score,
daily_study_time,
deadline
)
```
---
## (C) Curriculum
```sql
topics(
topic_id PK,
subject,
grade,
topic_name,
difficulty,
importance
)
```
---
## (D) Prerequisite Graph
```sql
topic_prerequisites(
topic_id FK,
prerequisite_id FK
)
```
---
## (E) Question Bank
```sql
questions(
```
